# Meta-SR
Meta-SR: A Magnification-Arbitrary Network for Super-Resolution(CVPR2019)(PyTorch)

Our code is built on [EDSR(PyTorch)](https://github.com/thstkdgus35/EDSR-PyTorch).
# Requirements

* Pytorch 0.4.0
* Python 3.5
* numpy
* skimage
* imageio
